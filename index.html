3D World Terrain

Developed detailed 3D terrain models using orthorectified imagery and elevation data to simulate real-world, geospecific terrain. The project involved processing GIS source data and generating 2D roads, 3D trees, 3D buildings, and 2D runway models, then rendering the environment in a proprietary gaming engine for simulation training purposes.
